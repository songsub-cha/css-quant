# 0008. ML 가격 예측 미도입

- **날짜**: 2026-07-11
- **상태**: Accepted
- **관련**: SoT [PART E1](../SoT.md#e1-adr-새-저장소-docsadr에-재기록) #8, [A4](../SoT.md#a4-mvp-정의--비목표), [A6.10](../SoT.md#a610-점수-모델-진화-경로-비-ml)

## 맥락

AI 종목 점수 파이프라인을 설계할 때, 팩터 기반 랭킹 대신 머신러닝으로 가격을 직접 예측하는
price oracle 방식을 검토할 수 있다. 그러나 일봉(daily) 가격 데이터는 신호 대 잡음비가 낮아, 가격을
직접 예측하는 모델은 백테스트에서 과적합되기 쉽고 재현성·설명 가능성도 떨어진다.

## 결정

**ML 가격 예측(price oracle)은 도입하지 않는다** — 비목표로 명시한다(A4).

## 결과

- 점수 모델은 팩터 기반 백분위 정규화 + 레짐별 가중 합산이라는 결정론적 방식을 유지한다 (A6.1).
- 점수 모델의 개선은 A6.10에 정의된 **비-ML 진화 경로**만을 따른다: 상시 IC(rank IC) 모니터링 →
  walk-forward 가중치 튜닝 → 단독 IC 검증을 통과한 팩터만 추가 → 섹터 중립화 → 레짐 4분화. 각 단계는
  명시적 도입 조건(예: IC 검증 통과, 백테스트 6개월+)을 만족해야 다음 단계로 진행한다.
- 원칙 7("숫자 계산은 결정론적 코드, 해석은 LLM")과 결합해, LLM 역시 점수 계산에 관여하지 않고 계산이
  끝난 결과의 한국어 해석만 생성한다 (A6.1).
- MVP 비목표에는 이와 함께 장중(intraday) 매매도 포함된다 — 일봉 기반 스윙만 지원하며 실시간 시세는
  감시/알림 용도로만 사용한다.
